# Coding assistants

Version: 1.30

On this page

Privatemode enables AI‑assisted coding on sensitive codebases that you can't share with public AI services. It provides an OpenAI‑compatible API, allowing you to connect any coding tool that supports such an API.

## Available models[​](#available-models "Direct link to Available models")

Privatemode offers the following models for coding tasks:

* **gpt-oss-120b**: a general‑purpose model with strong reasoning and code generation capabilities. ([Details](/models/gpt-oss-120b))
* **Qwen3‑Coder**: a fast, coding‑optimized model that supports tab‑completion and editing. ([Details](/models/qwen3-coder-30b-a3b))

## Supported IDEs and extensions[​](#supported-ides-and-extensions "Direct link to Supported IDEs and extensions")

Privatemode has been tested with these environments:

* OpenAI Codex ([Get started](/guides/coding-assistants-codex))
* Visual Studio Code with GitHub Copilot, Cline, and Continue extensions ([Get started](/guides/coding-assistants-vscode))
* JetBrains IDEs like IntelliJ IDEA and PyCharm with the JetBrains AI Assistant plugin ([Get started](/guides/coding-assistants-jetbrains))

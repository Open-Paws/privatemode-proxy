# Qwen3-Embedding-4B

Version: 1.30

On this page

The [Qwen3 Embedding 4b](https://huggingface.co/boboliu/Qwen3-Embedding-4B-W4A16-G128) model is a model from the Qwen family, specifically designed for text embedding tasks.
The model inherits the multilingual capabilities skills of its foundational model.

## Model ID[​](#model-id "Direct link to Model ID")

`qwen3-embedding-4b`

## Source[​](#source "Direct link to Source")

[Hugging face](https://huggingface.co/boboliu/Qwen3-Embedding-4B-W4A16-G128)

## Modality[​](#modality "Direct link to Modality")

* Input: text
* Output: embedding vector

## Context limit[​](#context-limit "Direct link to Context limit")

* Embedding size: Up to 2560, supports user-defined output dimensions ranging from 32 to 2560
* Maximum input size: 32000 tokens
